$.extend(目标对象，其他源对象...)
  • 简介：用于把源对象方法和属性扩展到目标对象上，只有一个入参，则扩展到jquery上。Jquery是类，$（“选择器”）是对象。
  • 简单的讲：就是继承。
  • 纯粹的对象：就是通过 { }、new Object()、Object.create(null) 创建的对象。
  • $.isPlainObject(val)：用于判断指定参数是否是一个纯粹的对象，返回值问boolean。
  • 数组对象.join("str")：指定str连接数组每一项，默认是逗号，。
  • $(function(){ }) 是 $(document).ready(function(){ })的简写，页面中所有内容加载完成才执行。相当于window.onload = function(){ }
  • DOM是什么？DOM就是一个html页面的标签树，成对不嫌。DOM加载完成了指页面所有的html标签（包括图片等）都加载完了，即浏览器已经响应完了，全部展现到浏览器界面上了。
