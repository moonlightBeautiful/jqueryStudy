首先，弄清楚什么是元素。元素：标签（标签包含标签属性和文本值）。节点：任何内容都是节点（标签，属性，文本，注释，空格等）

$("p1").contents()：获得选中元素所有子节点。

集合.filter(func(){ return boolean})： 返回符合一定条件的元素。

$("p1")..wrap("<元素>"):每个被选元素放置在指定的 HTML 元素中
